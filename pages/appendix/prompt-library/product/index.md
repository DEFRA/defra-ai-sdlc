# Product Prompts

This section contains prompts for product and requirements-related tasks.

## Available Prompts

- [API Requirements](prompt-api-requirements/) - Generate comprehensive API requirements and specifications
- [Architecture Decision Record](prompt-architecture-decision-record/) - Create structured architectural decision records
- [Combined Requirements Features Stories](prompt-combined-requirements-features-stories/) - Generate integrated requirements, features, and user stories
- [Data Model Generation](prompt-data-model-generation/) - Create data models and database schemas
- [High Level Architecture](prompt-high-level-architecture/) - Design high-level system architecture and components
- [Product Analysis](prompt-product-analysis/) - Conduct deep analysis of product requirements and effectiveness
- [Sequence Diagram](prompt-sequence-diagram/) - Generate sequence diagrams for system interactions
- [User Interface Requirements](prompt-user-interface-requirements/) - Define comprehensive UI/UX requirements
- [User Story Creation](prompt-user-story-creation/) - Create detailed user stories from requirements
